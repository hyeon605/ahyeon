# MentoringTalk
## Description
### Summary
멘토링제도를 이용하여 직업에 대한 방향성 및 궁금증 등을 해소할 수 있는 웹사이트<br/>
제작 기간 : 2023.06.19 ~ 진행중
<br/>
<br/>
<br/>
<br/>
#### 구현 기능
1. 회원 가입 및 간편 로그인
2. 멘토 리스트(멘토 찾기)
3. 온라인/전화/영상상담 예약
4. 멘토관리페이지
